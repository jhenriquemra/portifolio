Dominó para Tetraplégicos

Um sistema de controle por movimento de cabeça que permite a pessoas com tetraplegia jogar dominó de forma autônoma, utilizando Arduino Nano e sensor MPU-6050.

Arduino Nano MPU-6050 C/C++ Acessibilidade Hardware

Visão geral do projeto

🤗 impacto social
⚙ hardware + software
🏟 autonomia
🔭 sensor IMU

Sobre o projeto

O dominó é um dos jogos mais populares do Brasil. Está nas calçadas, nas mesas de bar, nas reuniões de família — é um ritual social enraizado na cultura latina. Mas infelizmente, pessoas com deficiências motoras são impedidas de jogar pela falta de acessibilidade.

Como funciona

[Descreva o funcionamento: o sensor MPU-6050 capta a inclinação e aceleração da cabeça do usuário; o Arduino Nano processa esses dados e os traduz em comandos de seleção e posicionamento das peças de dominó.]

Componentes utilizados

  1. Arduino Nano — Microcontrolador principal responsável por ler os dados do sensor e executar a lógica de controle.
  2. MPU-6050 — Sensor IMU (acelerômetro + giroscópio) que detecta a orientação e movimento da cabeça em 6 eixos.
  3. Jumpers — Conectam o sensor com o controlador.
  4. Tiara-Suporte — É a base para todos os Componentes e estrutura do projeto.

Desafios

Como as deficiências fisicas são muito variaveis, tivemos o desafio de fazer um produto que abrangisse todos. Por isso, buscamos criar algo que funcionasse no mais extremo das deficiências (Tetraplegia)

Aprendizados

Desenvolvi habilidades das mais variadas. Desde aprimorar meu C/C++, até como soldar peças e a logica de hardware. De fato um momento especial para minha caminhada
