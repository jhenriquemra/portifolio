## 📂 Estrutura do Repositório

O projeto é um site estático moderno, construído de forma limpa e estruturado da seguinte forma:

* **`index.html`**: Página principal (Home) com o meu perfil, visão geral dos meus interesses profissionais (como produtos digitais, dados e automação) e as competências que estou desenvolvendo.
* **`experiencia.html`**: Detalhamento da minha trajetória, incluindo passagens pelo mercado corporativo, conquistas em olimpíadas científicas e proficiência em idiomas.
* **`projeto.html`**: Página dedicada ao detalhamento do **DomiNod** (sistema de acessibilidade com hardware embarcado para tetraplégicos).
* **`links.html`**: Central de estudos e referências adicionais.
* **`style.css`**: Arquivo de estilização responsável pelo design responsivo, componentes visuais e identidade visual do portfólio.
* **`main.js`**: Scripts de interatividade (como o menu mobile e os gráficos/animações da página principal).
